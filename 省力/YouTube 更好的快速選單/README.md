# YouTube Better Quick Menu

## Problem / Requirement

YouTube hides the left sidebar mini-guide when watching videos, forcing users to click the hamburger menu every time they want to navigate. This script brings back the quick access sidebar on video watch pages.

### Features

- Hover mouse on the far left edge (within 100px) → Floating mini guide appears automatically
- Mouse leaves → Menu fades out smoothly
- Clean icons and labels, highly consistent with YouTube's native design
- SPA friendly – works when switching between videos
- Semi-transparent design to minimize interference with video content
- Multi-language support (Traditional Chinese, English, Japanese, Hindi, Czech, Lithuanian, etc.)

### Installation

1. Install [Violentmonkey](https://violentmonkey.github.io) or Tampermonkey
2. Install [YouTube Better Quick Menu](https://greasyfork.org/zh-TW/scripts/566549-youtube-%E6%9B%B4%E5%A5%BD%E7%9A%84%E5%BF%AB%E9%80%9F%E9%81%B8%E5%96%AE)
3. Refresh any YouTube video watch page

### Usage

- On any `/watch?v=` video page, move your mouse to the **extreme left side** of the screen.
- The floating quick menu will appear.
- Move mouse away to hide it.
- 在任何 `https://www.youtube.com/watch?v=` 的影片觀看頁面，將滑鼠遊標移到 **畫面最左側** 即可喚出浮動導覽選單。
- 滑鼠離開左側區域後，選單會自動隱藏。
- 點選選單中的項目即可快速切換頁面。
