# 文本轉連結 / Text Linkify

## 問題描述
許多網站的使用者撰寫的文本由於並非網站建立者建立或是網站不支援Markdown，網址並非可點擊連結而是純文本，除了多了一點麻煩以外也容易忽略。
而現有的類似腳本與瀏覽器擴充則有缺乏自訂性與浪費效能的問題。

### 功能說明
這個腳本會自動在網頁的**純文字內容**中辨識網址，並將其轉換成可點擊的 `<a>` 連結。

- 自動偵測 http/https 與 www. 開頭的網址
- 點擊後**背景新分頁**開啟（不跳離當前頁面）
- 支援**目前頁面網域黑名單**（使特定網站衝突時使用者有更簡單的辦法解決問題）
- 支援**網域別自訂掃描範圍**（只在特定區塊內轉換，例如只處理 .post-body）
- 支援**網域別自訂連結樣式**（預設為添加底線）
- 選單快速切換「將目前網域加入/移除黑名單」
- 多語言選單與提示（目前支援中英日西德印捷立等語言）
- 相容 Violentmonkey / Tampermonkey、Chrome / Firefox

### 安裝步驟

1. 安裝 [Violentmonkey](https://violentmonkey.github.io)（支援 Firefox、Chrome、Vivaldi 等瀏覽器）。
2. 安裝 [文本轉連結](https://greasyfork.org/zh-TW/scripts/567224-%E6%96%87%E6%9C%AC%E8%BD%89%E9%80%A3%E7%B5%90)
3. 完成

### 使用方法

- **基本使用**：安裝後無需任何設定，在大多數論壇、留言區自動轉換。
- **快速停用某網站**：點擊腳本圖示 → 選單 → 「將 xxx.com 加入黑名單」 → 頁面重新載入，腳本在此網站停用。

### 設定選項

| 選單項目                        | 功能說明                                      | 預設值          |
|--------------------------------|-----------------------------------------------|-----------------|
| 將 xxx.com 加入/移除黑名單      | 一鍵停用或啟用腳本於目前網站                  | —               |
| 設定此網域掃描範圍              | 輸入 CSS 選擇器，只在指定區域內轉換連結       | 整個頁面        |
| 設定此網域連結樣式              | 自訂 .auto-text-link 的 CSS（顏色、底線等）    | 預設藍底線      |

### 注意事項
- 極長單一文字節點（數十萬字以上）仍可能影響效能
- 黑名單支援子域名匹配（輸入 example.com 會同時擋住 www.example.com、sub.example.com）
